Plays a clip from Turn Down For What when publishing a post.

== Description ==

Do you need a bit more incentive to click the Publish button more often?
Do you want to scare yourself with random sounds coming from your WordPress admin suddenly?

This plugin is for you!  It's like Lil Jon is right inside your dashboard!

== Installation ==

1. Upload the folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
